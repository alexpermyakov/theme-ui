# Spinner

Spinning SVG loading indicator

```js
import { Spinner } from '@theme-ui/components'
```

```jsx live=true
<Spinner variant='primary' />
```

```jsx live=true
<Spinner
  variant='dualring' 
  duration={1200}
  item={{
    count: 4,
    delay: -150
  }}
/>
```

```jsx live=true
<Spinner
  variant='facebook'
  duration={1200}
  item={{
    count: 3,
    delay: 120,
    width: 12,
    height: 16,
    getX: (index, total, width) => (8 + 12) * index 
  }}
/>
```

```jsx live=true
<Spinner
  variant='roller'
  duration={1200}
  item={{
    count: 12,
    delay: 100,
    width: 6,
    height: 6,
    getX: (index, total, width) => {
      const r = width / 2;
      const angle = (index / total) * Math.PI * 2
      return r + r * Math.cos(angle)
    },
    getY: (index, total, height) => {
      const r = height / 2;
      const angle = (index / total) * Math.PI * 2
      return r + r * Math.sin(angle)
    }
  }}
/>
```

## Props

Name | Type | Description
---|---|---
`title` | String | (default `'loading'`) Text for SVG `<title>` tag
`size` | Number | (default `48`) chart diameter
`strokeWidth` | Number | (default `4`) stroke width

A `title` attribute should be provided to the component for accessibility purposes.
The element uses `role='img'` by default.
Pass any overrides or additional attributes for the SVG element as props.

## Variants

Spinner variants can be defined anywhere in the `theme` object.

```js
// example theme variants
{
  styles: {
    spinner: {
      color: 'red',
    },
  },
}
// <Spinner variant='styles.spinner' />
```
